for: Un bucle for se utiliza cuando quiero ejecutar una determinada acción para tantos elementos de una matriz como quiera. Debe definirse con un valor inicial, seguido de la condición que debe cumplirse para que el bucle se siga ejecutando, y por último, la iteración requerida. Por ejemplo, si quiero ejecutar una acción para todos los elementos de una matriz menos el primero, lo defino como sigue: for( let i = 1; i < array.length; i++){ acción a ejecutar comenzando en i = 1 y hasta que se deje de cumplir que i < array.length, iterando siempre al elemento siguiente del array }.
&&, ||, !: Estos símbolos son los operadores lógicos que se utilizan para definir condiciones. && ("y"), || ("o" inclusivo), ! (lo contrario a lo que le sigue).
